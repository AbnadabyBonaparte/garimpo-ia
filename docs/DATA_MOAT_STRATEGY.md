# 1. O que é Data Moat

Data moat é quando uma empresa possui um banco de dados que:

- concorrentes não possuem
- melhora continuamente
- cria vantagem acumulativa

Exemplos:

| Empresa | Data Moat |
|-------|------------|
Bloomberg | histórico financeiro global |
Zillow | dados imobiliários |
CoStar | dados imobiliários comerciais |
Kelley Blue Book | histórico de preços de veículos |
